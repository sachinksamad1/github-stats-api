# Security Policy

## Supported Versions

Currently, only the latest version of GitHub Stats API is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do not report it via public issues.

Instead, please report it by opening a private security advisory on GitHub or by contacting the maintainer directly.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.
